Леенсон Елизавета. Я вам всем покажу!.. Компания Microsoft слегка поступилась принципами
Компания Microsoft, выступающая против предоставления всем желающим тексты своих программ, фактически приступила к опубликованию доселе секретной информации. Пока этот ход можно рассматривать прежде всего как способ избегнуть серьезных финансовых потерь, поскольку 9 штатов и федеральный округ Колумбия до сих пор настаивают на более серьезных санкциях по отношению к монополисту.
Пока речь об открытии всех исходных текстов операционной системы Windows не идет. И вряд ли это произойдет даже в отдаленном будущем. Сейчас Microsoft на условиях лицензирования сделает доступной информацию о большом количестве коммуникационных протоколов различным компаниям-разработчикам ПО. Информация касается взаимодействию между Windows 2000 Professional, Windows XP и будущими версиями Windows. Теоретически эти данные должны облегчить компаниям-разработчикам создание своих программ, работающих в среде Windows. Однако, насколько добрая воля Microsoft повлияет на рынок ПО, сейчас не может сказать никто.
Следующий шаг компании должен быть еще более эффектным. Как сообщает сайт eWEEK, с 28 августа будет открыт доступ к большому количеству внутренних программных интерфейсов Internet Explorer, Windows Media Player, Windows Messenger, Outlook Express и Microsoft Java Virtual Machine. По крайней мере именно эти продукты указанны в мировом соглашении, подписанном в ноябре 2001 года между Microsoft и Департаментом юстиции США. eWEEK приводит слова вице-президента компании по юридическим и корпоративным вопросам Microsoft Брэда Смита, о том, что несмотря на ценность раскрываемых технологий, плата за их использование весьма мала.
Определенные изменения коснутся и обычных пользователей. Например с сентября этого года недоброжелатели продукции Microsoft смогут заменить в своем компьютере такие программы, как уже упомянутые Internet Explorer, Windows Media Player, Windows Messenger, Outlook Express и Microsoft Java Virtual Machine. Также этой возможностью смогут воспользоваться и производители компьютеров, предустанавливающие операционную систему Windows на свою продукцию. До сих пор лицензией на использование Windows такие действия запрещались.
Естественно, Билл Гейтс не был бы самим собой, если бы не использовал все описанные действия в качестве хорошей рекламной акции. Агентство Reuters сообщает, что компания начинает кампанию по разъяснению своих планов выполнения условий мирового соглашения. Причем в заявлении Microsoft особо подчеркивается, что компания делает это по своей воле, поскольку суд по этому поводу не высказывался.
На самом деле всеми этими действиями компания надеется отвести от себя более серьезное наказание. Стоит напомнить, что в июне федеральный апелляционный суд поддержал прежнее решение суда, признавшего некорректность рыночного поведения Microsoft. В частности было признано, что компания использовала свое монопольное положение на рынке операционных систем, интегрировав в Windows браузер Internet Explorer. В результате конкурирующая программа для работы с Интернетом производства компании Netscape была фактически загнана в рыночную резервацию. Сейчас Microsoft формально устранила все препятствия для своего конкурента, однако браузер Netscape (сейчас его выпускает Netscape Communication -- подразделение компании America Online) вряд ли сможет догнать Internet Explorer. Помня об этой истории, 9 штатов и федеральный округ Колумбия все еще предлагают наказать монополиста более жестоко.
